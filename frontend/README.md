# Frontend - App BiT (B2G)

Este directorio contendrá el código fuente de la Web App Responsiva (PWA) de la plataforma BiT, desarrollada utilizando **React y Tailwind CSS**.
La interfaz será la encargada de mostrar el mapa interactivo y la barra de AI Query para las consultas en lenguaje natural.

*Para más información sobre el proyecto, revisa el [README principal](../README.md).*
