# Backend - App BiT (B2G)

Este directorio contendrá el código fuente de la API y el procesamiento de datos, utilizando **JavaScript (Node.js)** y **Supabase (PostgreSQL)** como base de datos.
El backend será responsable de la ingestión del dataset Vísent CDRView, y de exponer los endpoints principales (`/datos` y `/mapa`) para que el frontend los consuma, incluyendo la integración con el agente de IA.

*Para más información sobre el proyecto, revisa el [README principal](../README.md).*
